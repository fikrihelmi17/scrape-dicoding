# What should be prepared?

**Python: 3.10.12**  
**Package Manager: pip**  
**Virtual Environment: venv**  
**Webdriver: Firefox (geckodriver)**  

===============================

# How to run the program?
`python3 -m venv .venv`  
`source .venv/bin/activate`   
`pip install -r requirement.txt`   
`chmod +x main.py`   
`./main.py`
